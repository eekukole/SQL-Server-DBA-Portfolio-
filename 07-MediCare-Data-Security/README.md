# Healthcare Data Security Implementation: Medicare Hospital



### Scenario:

I assumed the position of new DBA at Medicare Hospital with the current environment:

* Patient database with PHI (Protected Health Information)
* HIPAA compliance requirements
* Database backups sent to offsite storage
* Developers need to test with production-like data
* Customer service reps view patient info but shouldn't see SSNs
* Previous DBA left; databases being restored to new servers (orphaned users everywhere)



### My Mission:

Implement all three security layers to achieve compliance and solve operational problems.
