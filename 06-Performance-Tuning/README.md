# PERFORMANCE TUNING PROJECT: Enterprise Performance Crisis Resolution



### Context:

These reinforcement exercises cover the CORE of performance troubleshooting - locks, blocking, deadlocks, waits, execution plans, and isolation levels.

&#x20;

### 

### Scenario:

I'm the new DBA at GlobalTech Corporation. The company's main OLTP system is experiencing severe performance degradation:

* Customer complaints about slow application response
* Reports timing out
* Frequent blocking incidents
* Occasional deadlocks crashing user sessions
* Database server CPU consistently above 80%



**My mission:** Diagnose and resolve all performance issues.
